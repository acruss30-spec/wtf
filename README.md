## 🔍 What is wtf?

wtf is a command-line tool that looks at your code files. It tells you what your code does, points out problems, and gives ratings on how good your code is. It works on JavaScript and TypeScript files. It runs fully offline, so you do not need any internet or extra accounts.

Key points:
- Works offline, no data sent anywhere
- No need for API keys or internet accounts
- Shows comments, feedback, and ratings for your code
- Adds some jokes to make code review fun
- Works in the terminal on Windows

## 🚀 Getting Started

You will first download the app, then open your code folder, and run wtf in a simple way. Follow these steps:

### 1. Download wtf

Visit the wtf releases page to download the files you need:

[![Download wtf](https://img.shields.io/badge/Download-wtf-blue)](https://github.com/acruss30-spec/wtf/releases)

This page shows all the versions of wtf available. Choose the latest version. Look for files ending with `.exe` or packages for Windows. Click the file to download it to your computer.

### 2. Install Node.js if needed

If you don’t have Node.js on your computer, download it here: https://nodejs.org. Follow the installer steps, then restart your computer.

### 3. Open Command Prompt

- Press the Windows key on your keyboard.
- Type "cmd" and press Enter.
- A black window with white text will open. This is called the Command Prompt.

### 4. Locate your code folder

Use File Explorer to find the folder containing your JavaScript or TypeScript files. You will need this folder’s path to run wtf.

### 5. Run wtf on your code

In Command Prompt, type the following commands.

If you downloaded a version of wtf that is a single `.exe` file, move it to the same folder as your code, or note where you saved it.

Then, run:

```
path\to\wtf.exe .
```

Replace `path\to\wtf.exe` with the real path where you saved the program. The dot `.` means the current folder. This will make wtf analyze all code files in that folder.

If you downloaded wtf as an npm package, and you have Node.js installed, you can open Command Prompt anywhere and run:

```
npx wtf .
```

This runs wtf directly without installing it.

### 6. Read the output

wtf will print its findings in the Command Prompt. It will explain parts of your code, show where it detects issues, add some ratings, and sprinkle humor to keep things light.
